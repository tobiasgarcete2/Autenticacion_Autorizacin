# Autenticación y Autorización con Sesiones y JWT

Instalaciones:


Back-end con JWT:
```bash
cd backend-jwt
npm install
```

Back-end con Sesiones:
```bash
cd backend-session
npm install
```

Front-end:
```bash
cd frontend
npm install
```

Ejecución en Desarrollo:
Ingresar en cada directorio desde una terminal diferente y ejecutar el comando `npm run dev`
